еҫҲеӨҡдәәе–ңж¬ўйҖүжӢ©APPжҲ–зҪ‘з«ҷдёӯзҡ„ж·ұиүІжЁЎејҸпјҢд№ҹи®ёд»–д»¬жӣҙе–ңж¬ўиҝҷж ·зҡ„еӨ–и§ӮпјҢжҲ–иҖ…д»–д»¬жғіи®©иҮӘе·ұзҡ„зңјзқӣе…ҚеҸ—з–ІеҠігҖӮиҝҷзҜҮж–Үз« е°Ҷе‘ҠиҜүдҪ еҰӮдҪ•еңЁзҪ‘з«ҷдёӯе®һзҺ°дёҖдёӘиҮӘеҠЁзҡ„CSSж·ұиүІжЁЎејҸпјҢж №жҚ®и®ҝе®ўзҡ„зі»з»ҹдё»йўҳжқҘиҮӘеҠЁе“Қеә”гҖӮ

CSS ж·ұиүІжЁЎејҸ (Dark Mode)

еңЁ:rootж №е…ғзҙ дёӯе®ҡд№үеҸҳйҮҸжқҘи®ҫзҪ®дё»йўҳзҡ„йўңиүІгҖӮжҲ‘е»әи®®дҪ д№ҹиҝҷж ·еҒҡпјҢеӣ дёәиҝҷж ·дјҡдҪҝиҝҷдёӘиҝҮзЁӢе®№жҳ“еҫ—еӨҡгҖӮжҲ‘е®ҡд№үзҡ„еҸҳйҮҸеҰӮдёӢ:

:root иҝҷдёӘ CSS дјӘзұ»еҢ№й…Қж–ҮжЎЈж ‘зҡ„ж №е…ғзҙ гҖӮеҜ№дәҺ HTML жқҘиҜҙпјҢ**:root** иЎЁзӨә <html>е…ғзҙ пјҢйҷӨдәҶдјҳе…Ҳзә§жӣҙй«ҳд№ӢеӨ–пјҢдёҺ html йҖүжӢ©еҷЁзӣёеҗҢгҖӮ

еҰӮжһңеёҢжңӣеңЁж ·ејҸиЎЁдёӯдҪҝз”ЁиҝҷдәӣеҸҳйҮҸпјҢеҸҜд»Ҙиҝҷж ·еҒҡ

иҝҷж ·пјҢеҰӮжһңдҪ жғіж”№еҸҳдҪ зҡ„дё»йўҳйўңиүІпјҢжүҖйңҖиҰҒеҒҡзҡ„е°ұжҳҜдҝ®ж”№дҪ е®ҡд№үзҡ„еҸҳйҮҸпјҢжүҖжңүдҪҝз”ЁиҝҷдёӘеҸҳйҮҸзҡ„ж ·ејҸйғҪдјҡиў«жӣҙж–°гҖӮ

зҺ°еңЁжҲ‘д»¬йңҖиҰҒе®ҡд№үдёҖз»„ж–°зҡ„еҸҳйҮҸпјҢиҝҷдәӣеҸҳйҮҸе°ҶеңЁи°ғз”ЁCSSж·ұиүІжЁЎејҸж—¶дҪҝз”ЁгҖӮеҜ№жҲ‘жқҘиҜҙпјҢж·ұиүІжЁЎејҸзҡ„еҸҳйҮҸжҳҜиҝҷж ·зҡ„:

ж·»еҠ ж·ұиүІжЁЎејҸиҮӘеҠЁе“Қеә”

зҺ°еңЁжҲ‘д»¬е®ҡд№үдәҶдёӨз»„еҸҳйҮҸгҖӮеү©дёӢиҰҒеҒҡзҡ„дёҖд»¶дәӢе°ұжҳҜе°Ҷprefers-color-schemeеӘ’дҪ“жҹҘиҜўж·»еҠ еҲ°жҲ‘д»¬зҡ„ж·ұиүІжЁЎејҸеҸҳйҮҸдёӯгҖӮ
