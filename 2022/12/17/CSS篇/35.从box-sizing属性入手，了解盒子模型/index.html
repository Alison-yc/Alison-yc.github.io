д»Һbox-sizingеұһжҖ§е…ҘжүӢпјҢдәҶи§Јзӣ’еӯҗжЁЎеһӢ

иғҢжҷҜ

е…ҲеЈ°жҳҺдёҖдёӢиҝҗз”Ёзҡ„еңәжҷҜпјҢеҒҮеҰӮйЎ№зӣ®еёғеұҖдҪҝз”Ёзҡ„жҳҜиҮӘйҖӮеә”зҡ„еёғеұҖж–№ејҸпјҢdivз»ҷеҮәзҡ„е®ҪеәҰжҳҜзҷҫеҲҶжҜ”зҡ„еҪўејҸпјҢеҚіеҚ зӘ—еҸЈе®ҪеәҰзҡ„100%пјҢдҪҶиҫ№жЎҶborderе’ҢеҶ…иҫ№и·қpaddingжҳҜз”ЁеғҸзҙ жқҘиЎЁзӨәзҡ„пјҢжӯӨж—¶е°ұдјҡйҖ жҲҗдёҖдёӘй—®йўҳжҳҜdivжҖ»е®ҪеәҰи¶…иҝҮзӘ—еҸЈе®ҪеәҰгҖӮдёәдәҶйҒҝе…Қиҝҷз§Қй—®йўҳпјҢеҸҜд»ҘдҪҝз”ЁеұһжҖ§box-sizing:border-boxжқҘжҠҠ ж ҮеҮҶзӣ’жЁЎеһӢ еҸҳжҲҗ д»Јжӣҝ(IE)зӣ’жЁЎеһӢ пјҢд»ҺиҖҢдҪҝdivзҡ„жҖ»е®ҪеәҰдҫқз„¶жҳҜ100%

д»Җд№ҲжҳҜCSS зӣ’жЁЎеһӢ?

йЎөйқўеёғеұҖдёӯпјҢдёҖдёӘе…ғзҙ зҡ„еӨ–иҫ№и·қпјҲmarginпјүгҖҒ иҫ№жЎҶпјҲborderпјүгҖҒеҶ…иҫ№и·қпјҲpaddingпјүгҖҒеҶ…е®№пјҲcontentпјүз»„жҲҗдёҖдёӘзӣ’жЁЎеһӢгҖӮзӣ’жЁЎеһӢеҸҜеҲҶдёәж ҮеҮҶзӣ’жЁЎеһӢ е’Ң д»ЈжӣҝпјҲIEпјүзӣ’жЁЎеһӢгҖӮ

ж ҮеҮҶзӣ’жЁЎеһӢ

еңЁж ҮеҮҶжЁЎеһӢдёӯпјҢеҰӮжһңдҪ з»ҷзӣ’и®ҫзҪ® width е’Ң heightпјҢе®һйҷ…и®ҫзҪ®зҡ„жҳҜеҶ…е®№еҢәеҹҹ( content box)зҡ„е®Ҫй«ҳгҖӮ padding е’Ң border еҶҚеҠ дёҠи®ҫзҪ®зҡ„е®Ҫй«ҳдёҖиө·еҶіе®ҡж•ҙдёӘзӣ’еӯҗзҡ„еӨ§е°ҸгҖӮ

зӨәдҫӢпјҡ

еҰӮжһңдҪҝз”Ёж ҮеҮҶжЁЎеһӢпјҢе…ғзҙ жҖ»е®ҪеәҰ = 160px пјҲ100+25+25+5+5пјүпјҢжҖ»й«ҳеәҰ = 110px (50 + 25 + 25 + 5 + 5)пјҢеҚіеҶ…е®№еҢәеҹҹcontent boxеҠ  padding е’Ң border гҖӮ

жіЁ: margin дёҚи®Ўе…Ҙе®һйҷ…еӨ§е°Ҹ вҖ”вҖ” еҪ“з„¶пјҢе®ғдјҡеҪұе“Қзӣ’еӯҗеңЁйЎөйқўжүҖеҚ з©әй—ҙпјҢдҪҶжҳҜеҪұе“Қзҡ„жҳҜзӣ’еӯҗеӨ–йғЁз©әй—ҙгҖӮ

д»ЈжӣҝпјҲIEпјүзӣ’жЁЎеһӢ

дҪ еҸҜиғҪдјҡи®Өдёәзӣ’еӯҗзҡ„еӨ§е°ҸиҝҳиҰҒеҠ дёҠиҫ№жЎҶе’ҢеҶ…иҫ№и·қпјҢиҝҷж ·еҫҲйә»зғҰгҖӮ еӣ дёәиҝҷдёӘеҺҹеӣ пјҢcssиҝҳжңүдёҖдёӘжӣҝд»Јзӣ’жЁЎеһӢгҖӮдҪҝз”ЁиҝҷдёӘжЁЎеһӢпјҢжүҖжңүе®ҪеәҰйғҪжҳҜеҸҜи§Ғе®ҪеәҰпјҢжүҖд»ҘеҶ…е®№е®ҪеәҰжҳҜиҜҘе®ҪеәҰеҮҸеҺ»иҫ№жЎҶе’ҢеЎ«е……йғЁеҲҶгҖӮдҪҝз”ЁдёҠйқўзӣёеҗҢзҡ„ж ·ејҸеҫ—еҲ°жҖ»е®Ҫй«ҳжҳҜ width = 100px, height = 50pxгҖӮ

й»ҳи®ӨжөҸи§ҲеҷЁдјҡдҪҝз”Ёж ҮеҮҶжЁЎеһӢгҖӮеҰӮжһңйңҖиҰҒдҪҝз”Ёжӣҝд»ЈжЁЎеһӢпјҢжӮЁеҸҜд»ҘйҖҡиҝҮдёәе…¶и®ҫзҪ® box-sizing: border-box жқҘе®һзҺ°гҖӮ иҝҷж ·е°ұеҸҜд»Ҙе‘ҠиҜүжөҸи§ҲеҷЁдҪҝз”Ё border-box жқҘе®ҡд№үеҢәеҹҹпјҢд»ҺиҖҢи®ҫе®ҡжӮЁжғіиҰҒзҡ„еӨ§е°ҸгҖӮ

box-sizingеұһжҖ§

CSSдёӯзҡ„ box-sizing еұһжҖ§з”ЁдәҺе‘ҠиҜүжөҸи§ҲеҷЁеҰӮдҪ•и®Ўз®—дёҖдёӘе…ғзҙ жҳҜжҖ»е®ҪеәҰе’ҢжҖ»й«ҳеәҰ

еңЁ CSSзӣ’еӯҗжЁЎеһӢ зҡ„й»ҳи®Өе®ҡд№үйҮҢпјҢдҪ еҜ№дёҖдёӘе…ғзҙ жүҖи®ҫзҪ®зҡ„ width дёҺ height еҸӘдјҡеә”з”ЁеҲ°иҝҷдёӘе…ғзҙ зҡ„еҶ…е®№еҢәгҖӮеҰӮжһңиҝҷдёӘе…ғзҙ жңүд»»дҪ•зҡ„ border жҲ– padding пјҢз»ҳеҲ¶еҲ°еұҸе№•дёҠж—¶зҡ„зӣ’еӯҗе®ҪеәҰе’Ңй«ҳеәҰдјҡеҠ дёҠи®ҫзҪ®зҡ„иҫ№жЎҶе’ҢеҶ…иҫ№и·қеҖјгҖӮиҝҷж„Ҹе‘ізқҖеҪ“дҪ и°ғж•ҙдёҖдёӘе…ғзҙ зҡ„е®ҪеәҰе’Ңй«ҳеәҰж—¶йңҖиҰҒж—¶еҲ»жіЁж„ҸеҲ°иҝҷдёӘе…ғзҙ зҡ„иҫ№жЎҶе’ҢеҶ…иҫ№и·қгҖӮеҪ“жҲ‘д»¬е®һзҺ°е“Қеә”ејҸеёғеұҖж—¶пјҢиҝҷдёӘзү№зӮ№е°Өе…¶зғҰдәәгҖӮ

box-sizing еұһжҖ§еҸҜд»Ҙиў«з”ЁжқҘи°ғж•ҙиҝҷдәӣиЎЁзҺ°:

  • content-box жҳҜй»ҳи®ӨеҖјгҖӮеҰӮжһңдҪ и®ҫзҪ®дёҖдёӘе…ғзҙ зҡ„е®Ҫдёә100pxпјҢйӮЈд№ҲиҝҷдёӘе…ғзҙ зҡ„еҶ…е®№еҢәдјҡжңү100px е®ҪпјҢе№¶дё”д»»дҪ•иҫ№жЎҶе’ҢеҶ…иҫ№и·қзҡ„е®ҪеәҰйғҪдјҡиў«еўһеҠ еҲ°жңҖеҗҺз»ҳеҲ¶еҮәжқҘзҡ„е…ғзҙ е®ҪеәҰдёӯгҖӮ

    • е°әеҜёи®Ўз®—е…¬ејҸпјҡ

      width = еҶ…е®№зҡ„е®ҪеәҰ

      height = еҶ…е®№зҡ„й«ҳеәҰ

  • border-box е‘ҠиҜүжөҸи§ҲеҷЁпјҡдҪ жғіиҰҒи®ҫзҪ®зҡ„иҫ№жЎҶе’ҢеҶ…иҫ№и·қзҡ„еҖјжҳҜеҢ…еҗ«еңЁwidthеҶ…зҡ„гҖӮд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңдҪ е°ҶдёҖдёӘе…ғзҙ зҡ„widthи®ҫдёә100pxпјҢйӮЈд№Ҳиҝҷ100pxдјҡеҢ…еҗ«е®ғзҡ„borderе’ҢpaddingпјҢеҶ…е®№еҢәзҡ„е®һйҷ…е®ҪеәҰжҳҜwidthеҮҸеҺ»(border + padding)зҡ„еҖјгҖӮеӨ§еӨҡж•°жғ…еҶөдёӢпјҢиҝҷдҪҝеҫ—жҲ‘д»¬жӣҙе®№жҳ“ең°и®ҫе®ҡдёҖдёӘе…ғзҙ зҡ„е®Ҫй«ҳгҖӮ

    • е°әеҜёи®Ўз®—е…¬ејҸпјҡ

      width = border + padding + еҶ…е®№зҡ„е®ҪеәҰ

      height = border + padding + еҶ…е®№зҡ„й«ҳеәҰ
