01. JSXз®Җд»Ӣ

д»Җд№ҲжҳҜJSXпјҹ

д»Ҙзұ»дјјжЁЎжқҝиҜӯжі•зҡ„ж–№ејҸжҸҸиҝ°UIзҡ„дёҖз§ҚиҜӯжі•гҖӮе®ғжҳҜJSзҡ„иҜӯжі•зі–пјҢжң¬иҙЁдёҠд№ҹжҳҜJSгҖӮеҸҜд»ҘеңЁе…¶дёӯдҪҝз”ЁеҸҳйҮҸжҲ–иЎЁиҫҫејҸгҖҒеҮҪж•°зӯүгҖӮ

дёәд»Җд№ҲдҪҝз”ЁJSXпјҹ

е°Ҷж Үи®°дёҺйҖ»иҫ‘ е…ұеҗҢеӯҳж”ҫеңЁз§°д№ӢдёәвҖңз»„д»¶вҖқзҡ„жқҫж•ЈиҖҰеҗҲеҚ•е…ғд№ӢдёӯпјҢжқҘе®һзҺ°е…іжіЁзӮ№еҲҶзҰ»гҖӮ

ReactдёҚејәеҲ¶дҪҝз”ЁJSXпјҢдҪҶеңЁ JavaScript д»Јз Ғдёӯе°Ҷ JSX е’Ң UI ж”ҫеңЁдёҖиө·ж—¶пјҢдјҡеңЁи§Ҷи§үдёҠжңүиҫ…еҠ©дҪңз”ЁгҖӮ

еңЁ JSX дёӯеөҢе…ҘиЎЁиҫҫејҸ

еңЁ JSX иҜӯжі•дёӯпјҢдҪ еҸҜд»ҘеңЁеӨ§жӢ¬еҸ·еҶ…ж”ҫзҪ®д»»дҪ•жңүж•Ҳзҡ„ JavaScript иЎЁиҫҫејҸгҖӮ

JSX д№ҹжҳҜдёҖдёӘиЎЁиҫҫејҸ

еңЁзј–иҜ‘д№ӢеҗҺпјҢJSX иЎЁиҫҫејҸдјҡиў«иҪ¬дёәжҷ®йҖҡ JavaScript еҮҪж•°и°ғз”ЁпјҢе№¶дё”еҜ№е…¶еҸ–еҖјеҗҺеҫ—еҲ° JavaScript еҜ№иұЎ

еҸҜд»ҘжҠҠJSXеҪ“еҲқдёҖдёӘеҜ№иұЎжқҘеҜ№еҫ…пјҢеҸҜд»ҘдҪҝз”Ёе®ғиөӢеҖјз»ҷеҸҳйҮҸгҖҒеҪ“еҸӮж•°иҫ“е…ҘеҮҪж•°гҖҒиҝ”еӣһJSXзӯүгҖӮ

JSX зү№е®ҡеұһжҖ§

йҖҡиҝҮдҪҝз”Ёеј•еҸ·пјҢжқҘе°ҶеұһжҖ§еҖјжҢҮе®ҡдёәеӯ—з¬ҰдёІеӯ—йқўйҮҸпјҡ

JSXдёӯзҡ„еұһжҖ§еҗҚдҪҝз”ЁcamelCase(е°Ҹй©јеі°)е‘ҪеҗҚгҖӮеҰӮдёҠйқўдҫӢеӯҗдёӯзҡ„tabIndex

дҪҝз”ЁеӨ§жӢ¬еҸ·пјҢжқҘеңЁеұһжҖ§еҖјдёӯжҸ’е…ҘдёҖдёӘ JavaScript иЎЁиҫҫејҸпјҡ

дҪҝз”Ё JSX жҢҮе®ҡеӯҗе…ғзҙ 

JSX ж ҮзӯҫйҮҢиғҪеӨҹеҢ…еҗ«еҫҲеӨҡеӯҗе…ғзҙ :

JSX йҳІжӯўжіЁе…Ҙж”»еҮ»

React DOM еңЁжёІжҹ“жүҖжңүиҫ“е…ҘеҶ…е®№д№ӢеүҚпјҢй»ҳи®ӨдјҡиҝӣиЎҢиҪ¬д№үгҖӮе®ғеҸҜд»ҘзЎ®дҝқеңЁдҪ зҡ„еә”з”ЁдёӯпјҢж°ёиҝңдёҚдјҡжіЁе…ҘйӮЈдәӣе№¶йқһиҮӘе·ұжҳҺзЎ®зј–еҶҷзҡ„еҶ…е®№гҖӮжүҖжңүзҡ„еҶ…е®№еңЁжёІжҹ“д№ӢеүҚйғҪиў«иҪ¬жҚўжҲҗдәҶеӯ—з¬ҰдёІгҖӮиҝҷж ·еҸҜд»Ҙжңүж•Ҳең°йҳІжӯў XSSпјҲcross-site-scripting, и·Ёз«ҷи„ҡжң¬пјүж”»еҮ»гҖӮ
