JS 遍历方法

javascript 原生遍历方法的建议用法：

  • 用 for 循环遍历数组
  • 用 for-in 遍历对象
  • 用 for-of 遍历类数组对象（ES6）
  • 用 Object.keys()获取对象属性名的集合
