react дёӯ class з»„д»¶е’ҢеҮҪж•°з»„д»¶зҡ„еҢәеҲ«

зұ»з»„д»¶жҳҜдҪҝз”Ё ES6 зҡ„ classВ  жқҘе®ҡд№үзҡ„з»„д»¶гҖӮ еҮҪж•°з»„д»¶жҳҜжҺҘж”¶дёҖдёӘеҚ•дёҖзҡ„ В propsВ  еҜ№иұЎе№¶иҝ”еӣһдёҖдёӘ React е…ғзҙ гҖӮ

е…ідәҺ React зҡ„дёӨеҘ— APIпјҲзұ»пјҲclassпјүAPI е’ҢеҹәдәҺеҮҪж•°зҡ„й’©еӯҗпјҲhooksпјү APIпјүгҖӮе®ҳж–№жҺЁиҚҗдҪҝз”Ёй’©еӯҗпјҲеҮҪж•°пјүпјҢиҖҢдёҚжҳҜзұ»гҖӮеӣ дёәй’©еӯҗжӣҙз®ҖжҙҒпјҢд»Јз ҒйҮҸе°‘пјҢз”Ёиө·жқҘжҜ”иҫғвҖқиҪ»вҖқпјҢиҖҢзұ»жҜ”иҫғвҖқйҮҚвҖқгҖӮиҖҢдё”пјҢй’©еӯҗжҳҜеҮҪж•°пјҢжӣҙз¬ҰеҗҲ React еҮҪж•°ејҸзҡ„жң¬иҙЁгҖӮ

еҮҪж•°дёҖиҲ¬жқҘиҜҙпјҢеҸӘеә”иҜҘеҒҡдёҖд»¶дәӢпјҢе°ұжҳҜиҝ”еӣһдёҖдёӘеҖјгҖӮ еҰӮжһңдҪ жңүеӨҡдёӘж“ҚдҪңпјҢжҜҸдёӘж“ҚдҪңеә”иҜҘеҶҷжҲҗдёҖдёӘеҚ•зӢ¬зҡ„еҮҪж•°гҖӮиҖҢдё”пјҢж•°жҚ®зҡ„зҠ¶жҖҒеә”иҜҘдёҺж“ҚдҪңж–№жі•еҲҶзҰ»гҖӮж №жҚ®еҮҪж•°иҝҷз§ҚзҗҶеҝөпјҢReact зҡ„еҮҪж•°з»„д»¶еҸӘеә”иҜҘеҒҡдёҖд»¶дәӢжғ…пјҡиҝ”еӣһз»„д»¶зҡ„ HTML д»Јз ҒпјҢиҖҢжІЎжңүе…¶д»–зҡ„еҠҹиғҪгҖӮеҮҪж•°зҡ„иҝ”еӣһз»“жһңеҸӘдҫқиө–дәҺе®ғзҡ„еҸӮж•°гҖӮдёҚж”№еҸҳеҮҪж•°дҪ“еӨ–йғЁж•°жҚ®гҖҒеҮҪж•°жү§иЎҢиҝҮзЁӢйҮҢйқўжІЎжңүеүҜдҪңз”ЁгҖӮ

зұ»пјҲclassпјүжҳҜж•°жҚ®е’ҢйҖ»иҫ‘зҡ„е°ҒиЈ…гҖӮ д№ҹе°ұжҳҜиҜҙпјҢз»„д»¶зҡ„зҠ¶жҖҒе’Ңж“ҚдҪңж–№жі•жҳҜе°ҒиЈ…еңЁдёҖиө·зҡ„гҖӮеҰӮжһңйҖүжӢ©дәҶзұ»зҡ„еҶҷжі•пјҢе°ұеә”иҜҘжҠҠзӣёе…ізҡ„ж•°жҚ®е’Ңж“ҚдҪңпјҢйғҪеҶҷеңЁеҗҢдёҖдёӘ class йҮҢйқўгҖӮ

зұ»з»„д»¶

зјәзӮ№ :

  • еӨ§еһӢз»„д»¶еҫҲйҡҫжӢҶеҲҶе’ҢйҮҚжһ„пјҢд№ҹеҫҲйҡҫжөӢиҜ•гҖӮ
  • дёҡеҠЎйҖ»иҫ‘еҲҶж•ЈеңЁз»„д»¶зҡ„еҗ„дёӘж–№жі•д№ӢдёӯпјҢеҜјиҮҙйҮҚеӨҚйҖ»иҫ‘жҲ–е…іиҒ”йҖ»иҫ‘гҖӮ
  • з»„д»¶зұ»еј•е…ҘдәҶеӨҚжқӮзҡ„зј–зЁӢжЁЎејҸпјҢжҜ”еҰӮ render props е’Ңй«ҳйҳ¶з»„д»¶гҖӮ
  • йҡҫд»ҘзҗҶи§Јзҡ„ classпјҢзҗҶи§Ј JavaScript дёӯВ thisВ зҡ„е·ҘдҪңж–№ејҸгҖӮ

еҢәеҲ« пјҡ

  1. зҠ¶жҖҒзҡ„жңүж— 
  • hooksеҮәзҺ°д№ӢеүҚпјҢеҮҪж•°з»„д»¶жІЎжңүе®һдҫӢпјҢжІЎжңүз”ҹе‘Ҫе‘ЁжңҹпјҢжІЎжңүstateпјҢжІЎжңүthisпјҢжүҖд»ҘжҲ‘д»¬з§°еҮҪж•°з»„д»¶дёәж— зҠ¶жҖҒз»„д»¶гҖӮ hooksеҮәзҺ°д№ӢеүҚпјҢreactдёӯзҡ„еҮҪж•°з»„д»¶йҖҡеёёеҸӘиҖғиҷ‘иҙҹиҙЈUIзҡ„жёІжҹ“пјҢжІЎжңүиҮӘиә«зҡ„зҠ¶жҖҒжІЎжңүдёҡеҠЎйҖ»иҫ‘д»Јз ҒпјҢжҳҜдёҖдёӘзәҜеҮҪж•°гҖӮе®ғзҡ„иҫ“еҮәеҸӘз”ұеҸӮж•°propsеҶіе®ҡпјҢдёҚеҸ—е…¶д»–д»»дҪ•еӣ зҙ еҪұе“ҚгҖӮ
  1. и°ғз”Ёж–№ејҸзҡ„дёҚеҗҢ
  • еҮҪж•°з»„д»¶йҮҚж–°жёІжҹ“пјҢе°ҶйҮҚж–°и°ғз”Ёз»„д»¶ж–№жі•иҝ”еӣһж–°зҡ„reactе…ғзҙ гҖӮзұ»з»„д»¶йҮҚж–°жёІжҹ“е°ҶnewдёҖдёӘж–°зҡ„з»„д»¶е®һдҫӢпјҢз„¶еҗҺи°ғз”Ёrenderзұ»ж–№жі•иҝ”еӣһreactе…ғзҙ пјҢиҝҷд№ҹиҜҙжҳҺдёәд»Җд№Ҳзұ»з»„д»¶дёӯthisжҳҜеҸҜеҸҳзҡ„гҖӮ
  1. еӣ дёәи°ғз”Ёж–№ејҸдёҚеҗҢпјҢеңЁеҮҪж•°з»„д»¶дҪҝз”ЁдёӯдјҡеҮәзҺ°й—®йўҳ
  • еңЁж“ҚдҪңдёӯж”№еҸҳзҠ¶жҖҒеҖјпјҢзұ»з»„д»¶еҸҜд»ҘиҺ·еҸ–жңҖж–°зҡ„зҠ¶жҖҒеҖјпјҢиҖҢеҮҪж•°з»„д»¶еҲҷдјҡжҢүз…§йЎәеәҸиҝ”еӣһзҠ¶жҖҒеҖј
React HooksпјҲй’©еӯҗзҡ„дҪңз”Ёпјү

HookВ жҳҜ React 16.8 зҡ„ж–°еўһзү№жҖ§гҖӮе®ғеҸҜд»Ҙи®©дҪ еңЁдёҚзј–еҶҷ class зҡ„жғ…еҶөдёӢдҪҝз”Ё state д»ҘеҸҠе…¶д»–зҡ„ React зү№жҖ§гҖӮ

React Hooksзҡ„еҮ дёӘеёёз”Ёй’©еӯҗ:

  • useState() //зҠ¶жҖҒй’©еӯҗ

  • useContext() //е…ұдә«зҠ¶жҖҒй’©еӯҗ

  • useReducer() //action й’©еӯҗ

  • useEffect() //еүҜдҪңз”Ёй’©еӯҗ

    1. useCallback и®°еҝҶеҮҪж•° дёҖиҲ¬жҠҠеҮҪж•°ејҸз»„д»¶зҗҶи§Јдёәclassз»„д»¶renderеҮҪж•°зҡ„иҜӯжі•зі–пјҢжүҖд»ҘжҜҸж¬ЎйҮҚж–°жёІжҹ“зҡ„ж—¶еҖҷпјҢеҮҪж•°ејҸз»„д»¶еҶ…йғЁжүҖжңүзҡ„д»Јз ҒйғҪдјҡйҮҚж–°жү§иЎҢдёҖйҒҚгҖӮиҖҢжңүдәҶ useCallback е°ұдёҚдёҖж ·дәҶпјҢдҪ еҸҜд»ҘйҖҡиҝҮ useCallback иҺ·еҫ—дёҖдёӘи®°еҝҶеҗҺзҡ„еҮҪж•°гҖӮ
    1. useMemo и®°еҝҶз»„д»¶ useCallback зҡ„еҠҹиғҪе®Ңе…ЁеҸҜд»Ҙз”ұ useMemo жүҖеҸ–д»ЈпјҢеҰӮжһңдҪ жғійҖҡиҝҮдҪҝз”Ё useMemo иҝ”еӣһдёҖдёӘи®°еҝҶеҮҪж•°д№ҹжҳҜе®Ңе…ЁеҸҜд»Ҙзҡ„гҖӮ е”ҜдёҖзҡ„еҢәеҲ«жҳҜпјҡ useCallback дёҚдјҡжү§иЎҢз¬¬дёҖдёӘеҸӮж•°еҮҪж•°пјҢиҖҢжҳҜе°Ҷе®ғиҝ”еӣһз»ҷдҪ пјҢиҖҢ useMemo дјҡжү§иЎҢз¬¬дёҖдёӘеҮҪж•°е№¶дё”е°ҶеҮҪж•°жү§иЎҢз»“жһңиҝ”еӣһз»ҷдҪ гҖӮ

жүҖд»Ҙ useCallback еёёз”Ёи®°еҝҶдәӢд»¶еҮҪж•°пјҢз”ҹжҲҗи®°еҝҶеҗҺзҡ„дәӢд»¶еҮҪж•°е№¶дј йҖ’з»ҷеӯҗз»„д»¶дҪҝз”ЁгҖӮиҖҢ useMemo жӣҙйҖӮеҗҲз»ҸиҝҮеҮҪж•°и®Ўз®—еҫ—еҲ°дёҖдёӘзЎ®е®ҡзҡ„еҖјпјҢжҜ”еҰӮи®°еҝҶз»„д»¶гҖӮ

    1. useRef дҝқеӯҳеј•з”ЁеҖј
      useRef и·ҹ createRef зұ»дјјпјҢйғҪеҸҜд»Ҙз”ЁжқҘз”ҹжҲҗеҜ№ DOM еҜ№иұЎзҡ„еј•з”ЁгҖӮuseRef иҝ”еӣһзҡ„еҖјдј йҖ’з»ҷз»„д»¶жҲ–иҖ… DOM зҡ„ ref еұһжҖ§пјҢе°ұеҸҜд»ҘйҖҡиҝҮ ref.current еҖји®ҝй—®з»„д»¶жҲ–зңҹе®һзҡ„ DOM иҠӮзӮ№пјҢйҮҚзӮ№жҳҜз»„д»¶д№ҹжҳҜеҸҜд»Ҙи®ҝй—®еҲ°зҡ„пјҢд»ҺиҖҢеҸҜд»ҘеҜ№ DOM иҝӣиЎҢдёҖдәӣж“ҚдҪңпјҢжҜ”еҰӮзӣ‘еҗ¬дәӢд»¶зӯүзӯүгҖӮ
    1. useImperativeHandle з©ҝйҖҸ Ref

йҖҡиҝҮ useImperativeHandle з”ЁдәҺи®©зҲ¶з»„д»¶иҺ·еҸ–еӯҗз»„д»¶еҶ…зҡ„зҙўеј•

    1. useLayoutEffect еҗҢжӯҘжү§иЎҢеүҜдҪңз”Ё
      еӨ§йғЁеҲҶжғ…еҶөдёӢпјҢдҪҝз”Ё useEffect е°ұеҸҜд»Ҙеё®жҲ‘д»¬еӨ„зҗҶз»„д»¶зҡ„еүҜдҪңз”ЁпјҢдҪҶжҳҜеҰӮжһңжғіиҰҒеҗҢжӯҘи°ғз”ЁдёҖдәӣеүҜдҪңз”ЁпјҢжҜ”еҰӮеҜ№ DOM зҡ„ж“ҚдҪңпјҢе°ұйңҖиҰҒдҪҝз”Ё useLayoutEffectпјҢuseLayoutEffect дёӯзҡ„еүҜдҪңз”ЁдјҡеңЁ DOM жӣҙж–°д№ӢеҗҺеҗҢжӯҘжү§иЎҢгҖӮ

useEffectе’ҢuseLayoutEffectжңүд»Җд№ҲеҢәеҲ« пјҡз®ҖеҚ•жқҘиҜҙе°ұжҳҜи°ғз”Ёж—¶жңәдёҚеҗҢпјҢuseLayoutEffectе’ҢеҺҹжқҘcomponentDidMount&componentDidUpdateдёҖиҮҙпјҢеңЁreactе®ҢжҲҗDOMжӣҙж–°еҗҺй©¬дёҠеҗҢжӯҘи°ғз”Ёзҡ„д»Јз ҒпјҢдјҡйҳ»еЎһйЎөйқўжёІжҹ“гҖӮиҖҢuseEffectжҳҜдјҡеңЁж•ҙдёӘйЎөйқўжёІжҹ“е®ҢжүҚдјҡи°ғз”Ёзҡ„д»Јз ҒгҖӮе®ҳж–№е»әи®®дјҳе…ҲдҪҝз”ЁuseEffect
