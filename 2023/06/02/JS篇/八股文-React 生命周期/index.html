react-fiber

иғҢжҷҜ

react-fiber дә§з”ҹзҡ„ж №жң¬еҺҹеӣ пјҢжҳҜеӨ§йҮҸзҡ„еҗҢжӯҘи®Ўз®—д»»еҠЎйҳ»еЎһдәҶжөҸи§ҲеҷЁзҡ„ UI жёІжҹ“гҖӮй»ҳи®Өжғ…еҶөдёӢпјҢJS иҝҗз®—гҖҒйЎөйқўеёғеұҖе’ҢйЎөйқўз»ҳеҲ¶йғҪжҳҜиҝҗиЎҢеңЁжөҸи§ҲеҷЁзҡ„дё»зәҝзЁӢеҪ“дёӯпјҢд»–д»¬д№Ӣй—ҙжҳҜдә’ж–Ҙзҡ„е…ізі»гҖӮеҰӮжһң JS иҝҗз®—жҢҒз»ӯеҚ з”Ёдё»зәҝзЁӢпјҢйЎөйқўе°ұжІЎжі•еҫ—еҲ°еҸҠж—¶зҡ„жӣҙж–°гҖӮеҪ“жҲ‘д»¬и°ғз”ЁsetStateжӣҙж–°йЎөйқўзҡ„ж—¶еҖҷпјҢReact дјҡйҒҚеҺҶеә”з”Ёзҡ„жүҖжңүиҠӮзӮ№пјҢи®Ўз®—еҮәе·®ејӮпјҢз„¶еҗҺеҶҚжӣҙж–° UIгҖӮеҰӮжһңйЎөйқўе…ғзҙ еҫҲеӨҡпјҢж•ҙдёӘиҝҮзЁӢеҚ з”Ёзҡ„ж—¶жңәе°ұеҸҜиғҪи¶…иҝҮ 16 жҜ«з§’пјҢе°ұе®№жҳ“еҮәзҺ°жҺүеё§зҡ„зҺ°иұЎгҖӮ

е®һзҺ°еҺҹзҗҶ

  • react еҶ…йғЁиҝҗиҪ¬еҲҶдёүеұӮпјҡ

    • Virtual DOM еұӮпјҢжҸҸиҝ°йЎөйқўй•ҝд»Җд№Ҳж ·гҖӮ
    • Reconciler еұӮпјҢиҙҹиҙЈи°ғз”Ёз»„д»¶з”ҹе‘Ҫе‘Ёжңҹж–№жі•пјҢиҝӣиЎҢ Diff иҝҗз®—зӯүгҖӮ
    • Renderer еұӮпјҢж №жҚ®дёҚеҗҢзҡ„е№іеҸ°пјҢжёІжҹ“еҮәзӣёеә”зҡ„йЎөйқўпјҢжҜ”иҫғеёёи§Ғзҡ„жҳҜ ReactDOM е’Ң ReactNativeгҖӮ

Fiber е…¶е®һжҢҮзҡ„жҳҜдёҖз§Қж•°жҚ®з»“жһ„пјҢе®ғеҸҜд»Ҙз”ЁдёҖдёӘзәҜ JS еҜ№иұЎжқҘиЎЁзӨә пјҡ

  • дёәдәҶе®һзҺ°дёҚеҚЎйЎҝпјҢе°ұйңҖиҰҒжңүдёҖдёӘи°ғеәҰеҷЁ (Scheduler) жқҘиҝӣиЎҢд»»еҠЎеҲҶй…ҚгҖӮдјҳе…Ҳзә§й«ҳзҡ„д»»еҠЎпјҲеҰӮй”®зӣҳиҫ“е…ҘпјүеҸҜд»Ҙжү“ж–ӯдјҳе…Ҳзә§дҪҺзҡ„д»»еҠЎпјҲеҰӮ Diffпјүзҡ„жү§иЎҢпјҢд»ҺиҖҢжӣҙеҝ«зҡ„з”ҹж•ҲгҖӮд»»еҠЎзҡ„дјҳе…Ҳзә§жңүе…ӯз§Қпјҡ

    • synchronousпјҢдёҺд№ӢеүҚзҡ„ Stack Reconciler ж“ҚдҪңдёҖж ·пјҢеҗҢжӯҘжү§иЎҢ
    • taskпјҢеңЁ next tick д№ӢеүҚжү§иЎҢ
    • animationпјҢдёӢдёҖеё§д№ӢеүҚжү§иЎҢ
    • highпјҢеңЁдёҚд№…зҡ„е°ҶжқҘз«ӢеҚіжү§иЎҢ
    • lowпјҢзЁҚеҫ®е»¶иҝҹжү§иЎҢд№ҹжІЎе…ізі»
    • offscreenпјҢдёӢдёҖж¬Ў render ж—¶жҲ– scroll ж—¶жүҚжү§иЎҢ

  • Fiber ReconcilerпјҲreact пјүжү§иЎҢиҝҮзЁӢеҲҶдёә 2 дёӘйҳ¶ж®өпјҡ

    • йҳ¶ж®өдёҖпјҢз”ҹжҲҗ Fiber ж ‘пјҢеҫ—еҮәйңҖиҰҒжӣҙж–°зҡ„иҠӮзӮ№дҝЎжҒҜгҖӮиҝҷдёҖжӯҘжҳҜдёҖдёӘжёҗиҝӣзҡ„иҝҮзЁӢпјҢеҸҜд»Ҙиў«жү“ж–ӯгҖӮйҳ¶ж®өдёҖеҸҜиў«жү“ж–ӯзҡ„зү№жҖ§пјҢи®©дјҳе…Ҳзә§жӣҙй«ҳзҡ„д»»еҠЎе…Ҳжү§иЎҢпјҢд»ҺжЎҶжһ¶еұӮйқўеӨ§еӨ§йҷҚдҪҺдәҶйЎөйқўжҺүеё§зҡ„жҰӮзҺҮгҖӮ
    • йҳ¶ж®өдәҢпјҢе°ҶйңҖиҰҒжӣҙж–°зҡ„иҠӮзӮ№дёҖж¬ЎиҝҮжү№йҮҸжӣҙж–°пјҢиҝҷдёӘиҝҮзЁӢдёҚиғҪиў«жү“ж–ӯгҖӮ

  • Fiber ж ‘пјҡReact еңЁ render з¬¬дёҖж¬ЎжёІжҹ“ж—¶пјҢдјҡйҖҡиҝҮ React.createElement еҲӣе»әдёҖйў— Element ж ‘пјҢеҸҜд»Ҙз§°д№Ӣдёә Virtual DOM TreeпјҢз”ұдәҺиҰҒи®°еҪ•дёҠдёӢж–ҮдҝЎжҒҜпјҢеҠ е…ҘдәҶ FiberпјҢжҜҸдёҖдёӘ Element дјҡеҜ№еә”дёҖдёӘ Fiber NodeпјҢе°Ҷ Fiber Node й“ҫжҺҘиө·жқҘзҡ„з»“жһ„жҲҗдёә Fiber TreeгҖӮFiber Tree дёҖдёӘйҮҚиҰҒзҡ„зү№зӮ№жҳҜй“ҫиЎЁз»“жһ„пјҢе°ҶйҖ’еҪ’йҒҚеҺҶзј–зЁӢеҫӘзҺҜйҒҚеҺҶпјҢз„¶еҗҺй…ҚеҗҲ requestIdleCallback API, е®һзҺ°д»»еҠЎжӢҶеҲҶгҖҒдёӯж–ӯдёҺжҒўеӨҚгҖӮ
