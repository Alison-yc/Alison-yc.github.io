classзұ»

JSйҮҢзҡ„зұ»е°ұжҳҜжһ„йҖ еҮҪж•°зҡ„иҜӯжі•зі–

еҹәжң¬з”Ёжі•
  1. зұ»йҮҢйқўжңүдёӘconstructorеҮҪж•°,еҸҜд»ҘжҺҘж”¶дј йҖ’иҝҮжқҘзҡ„еҸӮж•°,еҗҢж—¶иҝ”еӣһе®һдҫӢеҜ№иұЎ
  2. constructorеҮҪж•°еҸӘиҰҒnewз”ҹжҲҗе®һдҫӢж—¶,е°ұдјҡиҮӘеҠЁи°ғз”ЁиҝҷдёӘеҮҪж•°,еҰӮжһңжҲ‘д»¬дёҚеҶҷиҝҷдёӘеҮҪж•°,зұ»д№ҹдјҡиҮӘеҠЁз”ҹжҲҗиҝҷдёӘеҮҪж•°
  3. е…¬е…ұеұһжҖ§ж”ҫеңЁconstructorдёӯпјҢе…¬е…ұж–№жі•зӣҙжҺҘеңЁзұ»йҮҢйқўеҶҷеҮҪж•°еЈ°жҳҺ,дјҡиҮӘеҠЁж·»еҠ иҮіеҺҹеһӢеҜ№иұЎдёӯ
  4. classзұ»жІЎжңүеҸҳйҮҸжҸҗеҚҮ,жүҖд»Ҙеҝ…йЎ»е…Ҳе®ҡд№үзұ»,жүҚиғҪйҖҡиҝҮзұ»е®һдҫӢеҢ–еҜ№иұЎ
  5. super()и°ғз”ЁзҲ¶зұ»йҮҢзҡ„constructorж–№жі•,еҸҜд»Ҙеҗ‘йҮҢйқўдј еҸӮ,е°ұзӯүдәҺFather.call(this,x,y)пјҢsuper еҝ…йЎ»еңЁеӯҗзұ»зҡ„thisеүҚйқўи°ғз”Ё
  6. classйҮҢйқўзҡ„ж–№жі•зҡ„thisжҢҮеҗ‘зҡ„жҳҜи°ғз”ЁиҖ…,еҰӮжһңи°ғз”ЁиҖ…дёҚжҳҜзұ»зҡ„е®һдҫӢ,е°ұйңҖиҰҒж”№еҸҳthisзҡ„жҢҮеҗ‘
е…ій”®иҜҚ

constructor
constructor()ж–№жі•жҳҜзұ»зҡ„й»ҳи®Өж–№жі•пјҢйҖҡиҝҮnewе‘Ҫд»Өз”ҹжҲҗеҜ№иұЎе®һдҫӢж—¶пјҢиҮӘеҠЁи°ғз”ЁиҜҘж–№жі•гҖӮдёҖдёӘзұ»еҝ…йЎ»жңүconstructor()ж–№жі•пјҢеҰӮжһңжІЎжңүжҳҫејҸе®ҡд№үпјҢдёҖдёӘз©әзҡ„constructor()ж–№жі•дјҡиў«й»ҳи®Өж·»еҠ гҖӮ

super

superиҝҷдёӘе…ій”®еӯ—пјҢж—ўеҸҜд»ҘеҪ“дҪңеҮҪж•°дҪҝз”ЁпјҢд№ҹеҸҜд»ҘеҪ“дҪңеҜ№иұЎдҪҝз”ЁгҖӮ
superдҪңдёәеҮҪж•°и°ғз”Ёж—¶пјҢд»ЈиЎЁзҲ¶зұ»зҡ„жһ„йҖ еҮҪж•°гҖӮеӯҗзұ»зҡ„жһ„йҖ еҮҪж•°еҝ…йЎ»жү§иЎҢдёҖж¬ЎsuperеҮҪж•°гҖӮsuperиҷҪз„¶д»ЈиЎЁдәҶзҲ¶зұ»Aзҡ„жһ„йҖ еҮҪж•°пјҢдҪҶжҳҜиҝ”еӣһзҡ„жҳҜеӯҗзұ»Bзҡ„е®һдҫӢ
superдҪңдёәеҜ№иұЎж—¶пјҢеңЁжҷ®йҖҡж–№жі•дёӯпјҢжҢҮеҗ‘зҲ¶зұ»зҡ„еҺҹеһӢеҜ№иұЎпјӣеңЁйқҷжҖҒж–№жі•дёӯпјҢжҢҮеҗ‘зҲ¶зұ»гҖӮ

getterгҖҒsetter
еңЁвҖңзұ»вҖқзҡ„еҶ…йғЁеҸҜд»ҘдҪҝз”Ёgetе’Ңsetе…ій”®еӯ—пјҢеҜ№жҹҗдёӘеұһжҖ§и®ҫзҪ®еӯҳеҖјеҮҪж•°е’ҢеҸ–еҖјеҮҪж•°пјҢжӢҰжҲӘиҜҘеұһжҖ§зҡ„еӯҳеҸ–иЎҢдёәгҖӮ

static
еҰӮжһңеңЁдёҖдёӘж–№жі•еүҚпјҢеҠ дёҠstaticе…ій”®еӯ—пјҢе°ұиЎЁзӨәиҜҘж–№жі•дёҚдјҡиў«е®һдҫӢз»§жүҝпјҢиҖҢжҳҜзӣҙжҺҘйҖҡиҝҮзұ»жқҘи°ғз”ЁпјҢиҝҷе°ұз§°дёәвҖңйқҷжҖҒж–№жі•вҖқгҖӮ
