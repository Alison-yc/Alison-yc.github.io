JS дёӯ this зҡ„дә”з§Қжғ…еҶө

  1. дҪңдёәжҷ®йҖҡеҮҪж•°жү§иЎҢж—¶пјҢthisжҢҮеҗ‘windowгҖӮ
  2. еҪ“еҮҪж•°дҪңдёәеҜ№иұЎзҡ„ж–№жі•иў«и°ғз”Ёж—¶пјҢthisе°ұдјҡжҢҮеҗ‘иҜҘеҜ№иұЎгҖӮ
  3. жһ„йҖ еҷЁи°ғз”ЁпјҢthis жҢҮеҗ‘иҝ”еӣһзҡ„иҝҷдёӘеҜ№иұЎгҖӮ
  4. з®ӯеӨҙеҮҪж•° з®ӯеӨҙеҮҪж•°зҡ„ this з»‘е®ҡзңӢзҡ„жҳҜthis жүҖеңЁеҮҪж•°е®ҡд№үеңЁе“ӘдёӘеҜ№иұЎдёӢпјҢе°ұз»‘е®ҡе“ӘдёӘеҜ№иұЎгҖӮеҰӮжһңжңүеөҢеҘ—зҡ„жғ…еҶөпјҢеҲҷ this з»‘е®ҡеҲ°жңҖиҝ‘зҡ„дёҖеұӮеҜ№иұЎдёҠгҖӮ
  5. еҹәдәҺ Function.prototype дёҠзҡ„ apply гҖҒ call е’Ң bind и°ғз”ЁжЁЎејҸпјҢиҝҷдёүдёӘж–№жі•йғҪеҸҜд»ҘжҳҫзӨәзҡ„жҢҮе®ҡи°ғз”ЁеҮҪж•°зҡ„ this жҢҮеҗ‘гҖӮapply жҺҘж”¶еҸӮж•°зҡ„жҳҜж•°з»„пјҢcall жҺҘеҸ—еҸӮж•°еҲ—иЎЁпјҢbind ж–№жі•йҖҡиҝҮдј е…ҘдёҖдёӘеҜ№иұЎпјҢиҝ”еӣһдёҖдёӘ this з»‘е®ҡдәҶдј е…ҘеҜ№иұЎзҡ„ж–°еҮҪж•°гҖӮиҝҷдёӘеҮҪж•°зҡ„ this жҢҮеҗ‘йҷӨдәҶдҪҝз”Ё new `ж—¶дјҡиў«ж”№еҸҳпјҢе…¶д»–жғ…еҶөдёӢйғҪдёҚдјҡж”№еҸҳгҖӮиӢҘдёәз©әй»ҳи®ӨжҳҜжҢҮеҗ‘е…ЁеұҖеҜ№иұЎ windowгҖӮ

еҺҹеһӢе…ізі»пјҡ

  • жҜҸдёӘ class йғҪжңүжҳҫзӨәеҺҹеһӢ prototype
  • жҜҸдёӘе®һдҫӢйғҪжңүйҡҗејҸеҺҹеһӢ _ proto_
  • е®һдҫӢзҡ„ proto жҢҮеҗ‘еҜ№еә” class зҡ„ prototype

вҖҢ еҺҹеһӢ: В  еңЁ JS дёӯпјҢжҜҸеҪ“е®ҡд№үдёҖдёӘеҜ№иұЎпјҲеҮҪж•°д№ҹжҳҜеҜ№иұЎпјүж—¶пјҢеҜ№иұЎдёӯйғҪдјҡеҢ…еҗ«дёҖдәӣйў„е®ҡд№үзҡ„еұһжҖ§гҖӮе…¶дёӯжҜҸдёӘеҮҪж•°еҜ№иұЎйғҪжңүдёҖдёӘprototypeВ  еұһжҖ§пјҢиҝҷдёӘеұһжҖ§жҢҮеҗ‘еҮҪж•°зҡ„еҺҹеһӢеҜ№иұЎгҖӮ

еҺҹеһӢй“ҫпјҡеҮҪж•°зҡ„еҺҹеһӢй“ҫеҜ№иұЎ constructor й»ҳи®ӨжҢҮеҗ‘еҮҪж•°жң¬иә«пјҢеҺҹеһӢеҜ№иұЎйҷӨдәҶжңүеҺҹеһӢеұһжҖ§еӨ–пјҢдёәдәҶе®һзҺ°з»§жүҝпјҢиҝҳжңүдёҖдёӘеҺҹеһӢй“ҫжҢҮй’Ҳproto,иҜҘжҢҮй’ҲжҳҜжҢҮеҗ‘дёҠдёҖеұӮзҡ„еҺҹеһӢеҜ№иұЎпјҢиҖҢдёҠдёҖеұӮзҡ„еҺҹеһӢеҜ№иұЎзҡ„з»“жһ„дҫқз„¶зұ»дјјгҖӮеӣ жӯӨеҸҜд»ҘеҲ©з”ЁprotoдёҖзӣҙжҢҮеҗ‘ Object зҡ„еҺҹеһӢеҜ№иұЎдёҠпјҢиҖҢ Object еҺҹеһӢеҜ№иұЎз”Ё Object.prototype.proto = null иЎЁзӨәеҺҹеһӢй“ҫйЎ¶з«ҜгҖӮеҰӮжӯӨеҪўжҲҗдәҶ js зҡ„еҺҹеһӢй“ҫз»§жүҝгҖӮеҗҢж—¶жүҖжңүзҡ„ js еҜ№иұЎйғҪжңү Object зҡ„еҹәжң¬ж–№жі•
зү№зӮ№: В JavaScript еҜ№иұЎжҳҜйҖҡиҝҮеј•з”ЁжқҘдј йҖ’зҡ„пјҢжҲ‘д»¬еҲӣе»әзҡ„жҜҸдёӘж–°еҜ№иұЎе®һдҪ“дёӯе№¶жІЎжңүдёҖд»ҪеұһдәҺиҮӘе·ұзҡ„еҺҹеһӢеүҜжң¬гҖӮеҪ“жҲ‘д»¬дҝ®ж”№еҺҹеһӢж—¶пјҢдёҺд№Ӣзӣёе…ізҡ„еҜ№иұЎд№ҹдјҡз»§жүҝиҝҷдёҖж”№еҸҳгҖӮ

newиҝҗз®—з¬Ұзҡ„е®һзҺ°жңәеҲ¶
  1. йҰ–е…ҲеҲӣе»әдәҶдёҖдёӘж–°зҡ„з©әеҜ№иұЎ
  2. и®ҫзҪ®еҺҹеһӢпјҢе°ҶеҜ№иұЎзҡ„еҺҹеһӢи®ҫзҪ®дёәеҮҪж•°зҡ„prototypeеҜ№иұЎгҖӮ
  3. и®©еҮҪж•°зҡ„thisжҢҮеҗ‘иҝҷдёӘеҜ№иұЎпјҢжү§иЎҢжһ„йҖ еҮҪж•°зҡ„д»Јз ҒпјҲдёәиҝҷдёӘж–°еҜ№иұЎж·»еҠ еұһжҖ§пјү
  4. еҲӨж–ӯеҮҪж•°зҡ„иҝ”еӣһеҖјзұ»еһӢпјҢеҰӮжһңжҳҜеҖјзұ»еһӢпјҢиҝ”еӣһеҲӣе»әзҡ„еҜ№иұЎгҖӮеҰӮжһңжҳҜеј•з”Ёзұ»еһӢпјҢе°ұиҝ”еӣһиҝҷдёӘеј•з”Ёзұ»еһӢзҡ„еҜ№иұЎгҖӮ
