йЎөйқўеёғеұҖ

  1. Flex еёғеұҖ

еёғеұҖзҡ„дј з»ҹи§ЈеҶіж–№жЎҲпјҢеҹәдәҺзӣ’зҠ¶жЁЎеһӢпјҢдҫқиө– display еұһжҖ§ + position еұһжҖ§ + float еұһжҖ§гҖӮе®ғеҜ№дәҺйӮЈдәӣзү№ж®ҠеёғеұҖйқһеёёдёҚж–№дҫҝпјҢжҜ”еҰӮпјҢеһӮзӣҙеұ…дёӯе°ұдёҚе®№жҳ“е®һзҺ°гҖӮ

Flex жҳҜ Flexible Box зҡ„зј©еҶҷпјҢж„ҸдёәвҖқеј№жҖ§еёғеұҖвҖқ,з”ЁжқҘдёәзӣ’зҠ¶жЁЎеһӢжҸҗдҫӣжңҖеӨ§зҡ„зҒөжҙ»жҖ§гҖӮжҢҮе®ҡе®№еҷЁ display: flex еҚіеҸҜгҖӮ з®ҖеҚ•зҡ„еҲҶдёәе®№еҷЁеұһжҖ§е’Ңе…ғзҙ еұһжҖ§гҖӮ

е®№еҷЁзҡ„еұһжҖ§пјҡ

  • flex-directionпјҡеҶіе®ҡдё»иҪҙзҡ„ж–№еҗ‘пјҲеҚіеӯҗ item зҡ„жҺ’еҲ—ж–№жі•пјүflex-direction: row | row-reverse | column | column-reverse;
  • flex-wrapпјҡеҶіе®ҡжҚўиЎҢи§„еҲҷ flex-wrap: nowrap | wrap | wrap-reverse;
  • flex-flowпјҡ .box { flex-flow: || ; }
  • justify-contentпјҡеҜ№е…¶ж–№ејҸпјҢж°ҙе№ідё»иҪҙеҜ№йҪҗж–№ејҸ
  • align-itemsпјҡеҜ№йҪҗж–№ејҸпјҢз«–зӣҙиҪҙзәҝж–№еҗ‘
  • align-content

йЎ№зӣ®зҡ„еұһжҖ§пјҲе…ғзҙ зҡ„еұһжҖ§пјүпјҡ

  • order еұһжҖ§пјҡе®ҡд№үйЎ№зӣ®зҡ„жҺ’еҲ—йЎәеәҸпјҢйЎәеәҸи¶Ҡе°ҸпјҢжҺ’еҲ—и¶Ҡйқ еүҚпјҢй»ҳи®Өдёә 0
  • flex-grow еұһжҖ§пјҡе®ҡд№үйЎ№зӣ®зҡ„ж”ҫеӨ§жҜ”дҫӢпјҢеҚідҪҝеӯҳеңЁз©әй—ҙпјҢд№ҹдёҚдјҡж”ҫеӨ§
  • flex-shrink еұһжҖ§пјҡе®ҡд№үдәҶйЎ№зӣ®зҡ„зј©е°ҸжҜ”дҫӢпјҢеҪ“з©әй—ҙдёҚи¶ізҡ„жғ…еҶөдёӢдјҡзӯүжҜ”дҫӢзҡ„зј©е°ҸпјҢеҰӮжһң е®ҡд№үдёӘ item зҡ„ flow-shrink дёә 0пјҢеҲҷдёәдёҚзј©е°Ҹ
  • flex-basis еұһжҖ§пјҡе®ҡд№үдәҶеңЁеҲҶй…ҚеӨҡдҪҷзҡ„з©әй—ҙпјҢйЎ№зӣ®еҚ жҚ®зҡ„з©әй—ҙгҖӮ
  • flexпјҡжҳҜ flex-grow е’Ң flex-shrinkгҖҒflex-basis зҡ„з®ҖеҶҷпјҢй»ҳи®ӨеҖјдёә 0 1 autoгҖӮ
  • align-selfпјҡе…Ғи®ёеҚ•дёӘйЎ№зӣ®дёҺе…¶д»–йЎ№зӣ®дёҚдёҖж ·зҡ„еҜ№йҪҗж–№ејҸпјҢеҸҜд»ҘиҰҶзӣ–
  • align-itemsпјҢй»ҳи®Өеұһ жҖ§дёә autoпјҢиЎЁзӨәз»§жүҝзҲ¶е…ғзҙ зҡ„ align-items жҜ”еҰӮиҜҙпјҢз”Ё flex е®һзҺ°еңЈжқҜеёғеұҖ
  1. Rem еёғеұҖ

йҰ–е…Ҳ Rem зӣёеҜ№дәҺж №(html)зҡ„ font-size еӨ§е°ҸжқҘи®Ўз®—гҖӮз®ҖеҚ•зҡ„иҜҙе®ғе°ұжҳҜдёҖдёӘзӣёеҜ№еҚ•дҫӢ еҰӮ:font-size:10px;,йӮЈд№ҲпјҲ1rem = 10pxпјүдәҶи§Ји®Ўз®—еҺҹзҗҶеҗҺйҰ–е…Ҳи§ЈеҶіжҖҺд№ҲеңЁдёҚеҗҢи®ҫеӨҮдёҠи®ҫзҪ® html зҡ„ font-size еӨ§е°ҸгҖӮе…¶е®һ rem еёғеұҖзҡ„жң¬иҙЁжҳҜзӯүжҜ”зј©ж”ҫпјҢдёҖиҲ¬жҳҜеҹәдәҺе®ҪеәҰгҖӮ

дјҳзӮ№пјҡеҸҜд»Ҙеҝ«йҖҹйҖӮз”Ёз§»еҠЁз«ҜеёғеұҖпјҢеӯ—дҪ“пјҢеӣҫзүҮй«ҳеәҰ
зјәзӮ№пјҡ
в‘  зӣ®еүҚ ie дёҚж”ҜжҢҒпјҢеҜ№ pc йЎөйқўжқҘи®ІдҪҝз”Ёж¬Ўж•°дёҚеӨҡпјӣ
в‘Ў ж•°жҚ®йҮҸеӨ§пјҡжүҖжңүзҡ„еӣҫзүҮпјҢзӣ’еӯҗйғҪйңҖиҰҒжҲ‘д»¬еҺ»з»ҷдёҖдёӘеҮҶзЎ®зҡ„еҖјпјӣжүҚиғҪдҝқиҜҒдёҚеҗҢжңәеһӢзҡ„йҖӮй…Қпјӣ
в‘ў еңЁе“Қеә”ејҸеёғеұҖдёӯпјҢеҝ…йЎ»йҖҡиҝҮ js жқҘеҠЁжҖҒжҺ§еҲ¶ж №е…ғзҙ  font-size зҡ„еӨ§е°ҸгҖӮд№ҹе°ұжҳҜиҜҙ css ж ·ејҸе’Ң js д»Јз ҒжңүдёҖе®ҡзҡ„иҖҰеҗҲжҖ§гҖӮдё”еҝ…йЎ»е°Ҷж”№еҸҳ font-size зҡ„д»Јз Ғж”ҫеңЁ css ж ·ејҸд№ӢеүҚгҖӮ

  1. зҷҫеҲҶжҜ”еёғеұҖ

йҖҡиҝҮзҷҫеҲҶжҜ”еҚ•дҪҚ вҖң % вҖң жқҘе®һзҺ°е“Қеә”ејҸзҡ„ж•ҲжһңгҖӮйҖҡиҝҮзҷҫеҲҶжҜ”еҚ•дҪҚеҸҜд»ҘдҪҝеҫ—жөҸи§ҲеҷЁдёӯзҡ„з»„д»¶зҡ„е®Ҫе’Ңй«ҳйҡҸзқҖжөҸи§ҲеҷЁзҡ„еҸҳеҢ–иҖҢеҸҳеҢ–пјҢд»ҺиҖҢе®һзҺ°е“Қеә”ејҸзҡ„ж•ҲжһңгҖӮ зӣҙи§Ӯзҡ„зҗҶи§ЈпјҢжҲ‘д»¬еҸҜиғҪдјҡи®Өдёәеӯҗе…ғзҙ зҡ„зҷҫеҲҶжҜ”е®Ңе…ЁзӣёеҜ№дәҺзӣҙжҺҘзҲ¶е…ғзҙ пјҢheight зҷҫеҲҶжҜ”зӣё еҜ№дәҺ heightпјҢwidth зҷҫеҲҶжҜ”зӣёеҜ№дәҺ widthгҖӮ paddingгҖҒborderгҖҒmargin зӯүзӯүдёҚи®әжҳҜеһӮзӣҙж–№еҗ‘иҝҳжҳҜж°ҙе№іж–№еҗ‘пјҢйғҪзӣёеҜ№дәҺзӣҙжҺҘзҲ¶е…ғзҙ зҡ„ widthгҖӮ йҷӨдәҶ border-radius еӨ–пјҢиҝҳжңүжҜ”еҰӮ translateгҖҒbackground-size зӯүйғҪжҳҜзӣёеҜ№дәҺиҮӘиә«зҡ„гҖӮ
