еүҚз«Ҝе®үе…ЁдәҶи§Јеҗ—пјҢиҜҙдёҖдёӢ XSS е’Ң CSRFпјҢд»ҘеҸҠжҖҺд№Ҳи§„йҒҝ

XSSпјҡи·Ёеҹҹи„ҡжң¬ж”»еҮ»

XSS ж”»еҮ»жҢҮзҡ„жҳҜи·Ёз«ҷи„ҡжң¬ж”»еҮ»пјҢжҳҜдёҖз§Қд»Јз ҒжіЁе…Ҙж”»еҮ»гҖӮж”»еҮ»иҖ…йҖҡиҝҮеңЁзҪ‘з«ҷжіЁе…ҘжҒ¶ж„Ҹи„ҡжң¬пјҢдҪҝд№ӢеңЁз”ЁжҲ·зҡ„жөҸи§ҲеҷЁдёҠиҝҗиЎҢпјҢд»ҺиҖҢзӣ—еҸ–з”ЁжҲ·зҡ„дҝЎжҒҜеҰӮ cookie зӯүгҖӮ
XSS зҡ„жң¬иҙЁжҳҜеӣ дёәзҪ‘з«ҷжІЎжңүеҜ№жҒ¶ж„Ҹд»Јз ҒиҝӣиЎҢиҝҮж»ӨпјҢдёҺжӯЈеёёзҡ„д»Јз Ғж··еҗҲеңЁдёҖиө·дәҶпјҢжөҸи§ҲеҷЁжІЎжңүеҠһжі•еҲҶиҫЁе“Әдәӣи„ҡжң¬жҳҜеҸҜдҝЎзҡ„пјҢд»ҺиҖҢеҜјиҮҙдәҶжҒ¶ж„Ҹд»Јз Ғзҡ„жү§иЎҢгҖӮ

ж”»еҮ»иҖ…еҸҜд»ҘйҖҡиҝҮиҝҷз§Қж”»еҮ»ж–№ејҸеҸҜд»ҘиҝӣиЎҢд»ҘдёӢж“ҚдҪңпјҡ

  • иҺ·еҸ–йЎөйқўзҡ„ж•°жҚ®пјҢеҰӮDOMгҖҒcookieгҖҒlocalStorageпјӣ
  • DOSж”»еҮ»пјҢеҸ‘йҖҒеҗҲзҗҶиҜ·жұӮпјҢеҚ з”ЁжңҚеҠЎеҷЁиө„жәҗпјҢд»ҺиҖҢдҪҝз”ЁжҲ·ж— жі•и®ҝй—®жңҚеҠЎеҷЁпјӣ
  • з ҙеқҸйЎөйқўз»“жһ„пјӣ
  • жөҒйҮҸеҠ«жҢҒпјҲе°Ҷй“ҫжҺҘжҢҮеҗ‘жҹҗзҪ‘з«ҷпјүпјӣ

йҳІеҫЎж–№жі•

  • еҸҜд»Ҙд»ҺжөҸи§ҲеҷЁзҡ„жү§иЎҢжқҘиҝӣиЎҢйў„йҳІпјҢдёҖз§ҚжҳҜдҪҝз”ЁзәҜеүҚз«Ҝзҡ„ж–№ејҸпјҢдёҚз”ЁжңҚеҠЎеҷЁз«ҜжӢјжҺҘеҗҺиҝ”еӣһпјҲдёҚдҪҝз”ЁжңҚеҠЎз«ҜжёІжҹ“пјүгҖӮеҸҰдёҖз§ҚжҳҜеҜ№йңҖиҰҒжҸ’е…ҘеҲ° HTML дёӯзҡ„д»Јз ҒеҒҡеҘҪе……еҲҶзҡ„иҪ¬д№үгҖӮеҜ№дәҺ DOM еһӢзҡ„ж”»еҮ»пјҢдё»иҰҒжҳҜеүҚз«Ҝи„ҡжң¬зҡ„дёҚеҸҜйқ иҖҢйҖ жҲҗзҡ„пјҢеҜ№дәҺж•°жҚ®иҺ·еҸ–жёІжҹ“е’Ңеӯ—з¬ҰдёІжӢјжҺҘзҡ„ж—¶еҖҷеә”иҜҘеҜ№еҸҜиғҪеҮәзҺ°зҡ„жҒ¶ж„Ҹд»Јз Ғжғ…еҶөиҝӣиЎҢеҲӨж–ӯгҖӮ

CSRFпјҡи·Ёз«ҷиҜ·жұӮдјӘйҖ 

CSRF ж”»еҮ»жҢҮзҡ„жҳҜи·Ёз«ҷиҜ·жұӮдјӘйҖ ж”»еҮ»пјҢж”»еҮ»иҖ…иҜұеҜјз”ЁжҲ·иҝӣе…ҘдёҖдёӘз¬¬дёүж–№зҪ‘з«ҷпјҢз„¶еҗҺиҜҘзҪ‘з«ҷеҗ‘иў«ж”»еҮ»зҪ‘з«ҷеҸ‘йҖҒи·Ёз«ҷиҜ·жұӮгҖӮеҰӮжһңз”ЁжҲ·еңЁиў«ж”»еҮ»зҪ‘з«ҷдёӯдҝқеӯҳдәҶзҷ»еҪ•зҠ¶жҖҒпјҢйӮЈд№Ҳж”»еҮ»иҖ…е°ұеҸҜд»ҘеҲ©з”ЁиҝҷдёӘзҷ»еҪ•зҠ¶жҖҒпјҢз»•иҝҮеҗҺеҸ°зҡ„з”ЁжҲ·йӘҢиҜҒпјҢеҶ’е……з”ЁжҲ·еҗ‘жңҚеҠЎеҷЁжү§иЎҢдёҖдәӣж“ҚдҪңгҖӮ
CSRF ж”»еҮ»зҡ„жң¬иҙЁжҳҜеҲ©з”Ё cookie дјҡеңЁеҗҢжәҗиҜ·жұӮдёӯжҗәеёҰеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁзҡ„зү№зӮ№пјҢд»ҘжӯӨжқҘе®һзҺ°з”ЁжҲ·зҡ„еҶ’е……гҖӮ

CSRF ж”»еҮ»еҸҜд»ҘдҪҝз”Ёд»ҘдёӢж–№жі•жқҘйҳІжҠӨпјҡ

  • иҝӣиЎҢеҗҢжәҗжЈҖжөӢпјҢжңҚеҠЎеҷЁж №жҚ® http иҜ·жұӮеӨҙдёӯ origin жҲ–иҖ… referer дҝЎжҒҜжқҘеҲӨж–ӯиҜ·жұӮжҳҜеҗҰдёәе…Ғи®ёи®ҝй—®зҡ„з«ҷзӮ№пјҢд»ҺиҖҢеҜ№иҜ·жұӮиҝӣиЎҢиҝҮж»ӨгҖӮ
  • дҪҝз”Ё CSRF Token иҝӣиЎҢйӘҢиҜҒпјҢжңҚеҠЎеҷЁеҗ‘з”ЁжҲ·иҝ”еӣһдёҖдёӘйҡҸжңәж•° Token пјҢеҪ“зҪ‘з«ҷеҶҚж¬ЎеҸ‘иө·иҜ·жұӮж—¶пјҢеңЁиҜ·жұӮеҸӮж•°дёӯеҠ е…ҘжңҚеҠЎеҷЁз«Ҝиҝ”еӣһзҡ„ token пјҢз„¶еҗҺжңҚеҠЎеҷЁеҜ№иҝҷдёӘ token иҝӣиЎҢйӘҢиҜҒгҖӮ
  • еҜ№ Cookie иҝӣиЎҢеҸҢйҮҚйӘҢиҜҒпјҢжңҚеҠЎеҷЁеңЁз”ЁжҲ·и®ҝй—®зҪ‘з«ҷйЎөйқўж—¶пјҢеҗ‘иҜ·жұӮеҹҹеҗҚжіЁе…ҘдёҖдёӘCookieпјҢеҶ…е®№дёәйҡҸжңәеӯ—з¬ҰдёІпјҢз„¶еҗҺеҪ“з”ЁжҲ·еҶҚж¬Ўеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒиҜ·жұӮзҡ„ж—¶еҖҷпјҢд»Һ cookie дёӯеҸ–еҮәиҝҷдёӘеӯ—з¬ҰдёІпјҢж·»еҠ еҲ° URL еҸӮж•°дёӯпјҢз„¶еҗҺжңҚеҠЎеҷЁйҖҡиҝҮеҜ№ cookie дёӯзҡ„ж•°жҚ®е’ҢеҸӮж•°дёӯзҡ„ж•°жҚ®иҝӣиЎҢжҜ”иҫғпјҢжқҘиҝӣиЎҢйӘҢиҜҒгҖӮ
  • еңЁи®ҫзҪ® cookie еұһжҖ§зҡ„ж—¶еҖҷи®ҫзҪ® Samesite пјҢйҷҗеҲ¶ cookie дёҚиғҪдҪңдёәиў«з¬¬дёүж–№дҪҝз”ЁпјҢд»ҺиҖҢеҸҜд»ҘйҒҝе…Қиў«ж”»еҮ»иҖ…еҲ©з”ЁгҖӮSamesite дёҖе…ұжңүдёӨз§ҚжЁЎејҸпјҢдёҖз§ҚжҳҜдёҘж јжЁЎејҸпјҢеңЁдёҘж јжЁЎејҸдёӢ cookie еңЁд»»дҪ•жғ…еҶөдёӢйғҪдёҚеҸҜиғҪдҪңдёәз¬¬дёүж–№ Cookie дҪҝз”ЁпјҢеңЁе®ҪжқҫжЁЎејҸдёӢпјҢcookie еҸҜд»Ҙиў«иҜ·жұӮжҳҜ GET иҜ·жұӮпјҢдё”дјҡеҸ‘з”ҹйЎөйқўи·іиҪ¬зҡ„иҜ·жұӮжүҖдҪҝз”ЁгҖӮ

д»Җд№ҲжҳҜдёӯй—ҙдәәж”»еҮ»пјҹ

дёӯй—ҙвјҲ (Man-in-the-middle attack, MITM) жҳҜжҢҮж”»еҮ»иҖ…дёҺйҖҡи®Ҝзҡ„дёӨз«ҜеҲҶеҲ«еҲӣе»әзӢ¬вҪҙзҡ„иҒ”зі», е№¶дәӨжҚўе…¶жүҖж”¶еҲ°зҡ„ж•°жҚ®, дҪҝйҖҡи®Ҝзҡ„дёӨз«Ҝи®Өдёәд»–д»¬жӯЈеңЁйҖҡиҝҮвјҖдёӘз§ҒеҜҶзҡ„иҝһжҺҘдёҺеҜ№вҪ…зӣҙжҺҘеҜ№иҜқ, дҪҶдәӢе®һдёҠж•ҙдёӘдјҡиҜқйғҪиў«ж”»еҮ»иҖ…е®Ңе…ЁжҺ§еҲ¶гҖӮеңЁдёӯй—ҙвјҲж”»еҮ»дёӯпјҢж”»еҮ»иҖ…еҸҜд»ҘжӢҰжҲӘйҖҡи®ҜеҸҢвҪ…зҡ„йҖҡиҜқе№¶жҸ’вјҠж–°зҡ„еҶ…е®№гҖӮ

ж”»еҮ»иҝҮзЁӢеҰӮдёӢ:

  • е®ўжҲ·з«ҜеҸ‘йҖҒиҜ·жұӮеҲ°жңҚеҠЎз«ҜпјҢиҜ·жұӮиў«дёӯй—ҙвјҲжҲӘиҺ·
  • жңҚеҠЎеҷЁеҗ‘е®ўжҲ·з«ҜеҸ‘йҖҒе…¬й’Ҙ
  • дёӯй—ҙвјҲжҲӘиҺ·е…¬й’ҘпјҢдҝқз•ҷеңЁвҫғвј°вјҝдёҠгҖӮз„¶еҗҺвҫғвј°вҪЈжҲҗвјҖдёӘдјӘйҖ зҡ„е…¬й’ҘпјҢеҸ‘з»ҷе®ўжҲ·з«Ҝ
  • е®ўжҲ·з«Ҝж”¶еҲ°дјӘйҖ зҡ„е…¬й’ҘеҗҺпјҢвҪЈжҲҗеҠ еҜҶhashеҖјеҸ‘з»ҷжңҚеҠЎеҷЁ
  • дёӯй—ҙвјҲиҺ·еҫ—еҠ еҜҶhashеҖјпјҢвҪӨвҫғвј°зҡ„з§Ғй’Ҙи§ЈеҜҶиҺ·еҫ—зңҹз§ҳй’Ҙ,еҗҢж—¶вҪЈжҲҗеҒҮзҡ„еҠ еҜҶhashеҖјпјҢеҸ‘з»ҷжңҚеҠЎеҷЁ
  • жңҚеҠЎеҷЁвҪӨз§Ғй’Ҙи§ЈеҜҶиҺ·еҫ—еҒҮеҜҶй’Ҙ,з„¶еҗҺеҠ еҜҶж•°жҚ®дј иҫ“з»ҷе®ўжҲ·з«Ҝ
