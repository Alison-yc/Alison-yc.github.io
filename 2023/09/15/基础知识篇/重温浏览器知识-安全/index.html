С╗ђС╣ѕтЈ»УЃйт╝ЋУхитЅЇуФ»т«ЅтЁежЌ«жбў№╝Ъ

  • УиеуФЎУёџТюг (Cross-Site Scripting, XSS): Р╝ђуДЇС╗БуаЂТ│еР╝іРйЁт╝Ј, СИ║С║єСИј CSS тї║тѕєТЅђС╗ЦУбФуД░Сйю XSSсђѓТЌЕТюЪтИИР╗ЁС║јРй╣у╗юУ«║тЮЏ, УхитЏаТў»Рй╣уФЎТ▓АТюЅт»╣РйцТѕиуџёУЙЊР╝іУ┐ЏРЙЈСИЦТа╝уџёжЎљтѕХ, Сй┐тЙЌТћ╗тЄ╗УђЁтЈ»С╗Цт░єУёџТюгСИіС╝атѕ░тИќР╝дУ«ЕтЁХС╗ќР╝ѕТхЈУДѕтѕ░ТюЅТЂХТёЈУёџТюгуџёР╗џРЙ», тЁХТ│еР╝іРйЁт╝ЈтЙѕу«ђтЇЋтїЁТІгСйєСИЇжЎљС║ј JavaScript / CSS / Flash уГЅ№╝Џ
  • iframeуџёТ╗ЦРйц: iframeСИГуџётєЁт«╣Тў»ућ▒уггСИЅРйЁТЮЦТЈљСЙЏуџё№╝їж╗ўУ«цТЃЁтєхСИІС╗ќС╗гСИЇтЈЌТјДтѕХ№╝їС╗ќС╗гтЈ»С╗ЦтюеiframeСИГУ┐љРЙЈJavaScirptУёџТюгсђЂFlashТЈњС╗ХсђЂт╝╣тЄ║т»╣У»ЮТАєуГЅуГЅ№╝їУ┐ЎтЈ»УЃйС╝џуа┤тЮЈтЅЇуФ»РйцТѕиСйЊжфї№╝Џ
  • УиеуФЎуѓ╣У»иТ▒ѓС╝фжђа№╝ѕCross-Site Request Forgeries№╝їCSRF№╝Ѕ: ТїЄТћ╗тЄ╗УђЁжђџУ┐ЄУ«Йуй«тЦйуџёжЎижў▒№╝їт╝║тѕХт»╣ти▓т«їТѕљУ«цУ»ЂуџёРйцТѕиУ┐ЏРЙЈРЙ«жбёТюЪуџёСИфР╝ѕС┐АТЂ»ТѕќУ«Йт«џС┐АТЂ»уГЅТЪљС║ЏуіХТђЂТЏ┤Тќ░№╝їт▒ъС║јУбФтіеТћ╗тЄ╗
  • ТЂХТёЈуггСИЅРйЁт║Њ: РйєУ«║Тў»тљјуФ»ТюЇтіАтЎет║ћРйцУ┐ўТў»тЅЇуФ»т║ћРйцт╝ђтЈЉ№╝їу╗ЮР╝цтцџТЋ░ТЌХтђЎжЃйТў»тюетђЪтіЕт╝ђтЈЉТАєТъХтњїтљёуДЇу▒╗т║ЊУ┐ЏРЙЈт┐ФжђЪт╝ђтЈЉ№╝їР╝ђТЌдуггСИЅРйЁт║ЊУбФТцЇР╝іТЂХТёЈС╗БуаЂтЙѕт«╣ТўЊт╝ЋУхит«ЅтЁежЌ«жбўсђѓ

уйЉу╗ютіФТїЂТюЅтЊфтЄауДЇ№╝їтдѓСйЋжў▓УїЃ№╝Ъ

Рй╣у╗ютіФТїЂтѕєСИ║СИцуДЇ:

  1. DNSтіФТїЂ: (УЙЊР╝іС║гСИюУбФт╝║тѕХУи│Уйгтѕ░Тиўт«ЮУ┐Ўт░▒т▒ъС║јdnsтіФТїЂ)

    • DNSт╝║тѕХУДБТъљ: жђџУ┐ЄС┐«Тћ╣У┐љУљЦтЋєуџёТюгтю░DNSУ«░тйЋ№╝їТЮЦт╝Ћт»╝РйцТѕиТхЂжЄЈтѕ░у╝ЊтГўТюЇтіАтЎе
    • 302Уи│УйгуџёРйЁт╝Ј: жђџУ┐ЄуЏЉТјДРй╣у╗ютЄ║Р╝ЮуџёТхЂжЄЈ№╝їтѕєТъљтѕцТќГтЊфС║ЏтєЁт«╣Тў»тЈ»С╗ЦУ┐ЏРЙЈтіФТїЂтцёуљєуџё,тєЇт»╣тіФТїЂуџётєЁтГўтЈЉУхи302Уи│УйгуџётЏътцЇ№╝їт╝Ћт»╝РйцТѕиУјитЈќтєЁт«╣

  2. HTTPтіФТїЂ: (У«┐жЌ«РЙЋТГїСйєТў»Р╝ђуЏ┤ТюЅУ┤фујЕУЊЮРйЅуџёР╝┤тЉі),ућ▒С║јhttpТўјРйѓС╝аУЙЊ,У┐љУљЦтЋєС╝џС┐«Тћ╣СйауџёhttpтЊЇт║ћтєЁт«╣(тЇ│тіаР╝┤тЉі)

    DNSтіФТїЂућ▒С║јТХЅтФїУ┐ЮТ│Ћ№╝їти▓у╗ЈУбФуЏЉу«АУхиТЮЦ№╝їуј░тюетЙѕт░ЉС╝џТюЅDNSтіФТїЂ№╝їРййhttpтіФТїЂСЙЮуёХРЙ«тИИуЏЏРЙЈ№╝їТюђТюЅТЋѕуџётіъТ│Ћт░▒Тў»тЁеуФЎHTTPS№╝їт░єHTTPтіат»є№╝їУ┐ЎСй┐тЙЌУ┐љУљЦтЋєРйєТ│ЋУјитЈќТўјРйѓ№╝їт░▒РйєТ│ЋтіФТїЂСйауџётЊЇт║ћтєЁт«╣сђѓ
