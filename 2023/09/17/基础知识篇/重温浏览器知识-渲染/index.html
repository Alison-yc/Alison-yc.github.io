JavaScript зҡ„еҠ иҪҪгҖҒи§ЈжһҗдёҺжү§иЎҢдјҡйҳ»еЎһж–ҮжЎЈзҡ„и§ЈжһҗпјҢд№ҹе°ұжҳҜиҜҙпјҢеңЁжһ„е»ә DOM ж—¶пјҢHTML и§ЈжһҗеҷЁиӢҘйҒҮеҲ°дәҶ JavaScriptпјҢйӮЈд№Ҳе®ғдјҡжҡӮеҒңж–ҮжЎЈзҡ„и§ЈжһҗпјҢе°ҶжҺ§еҲ¶жқғз§»дәӨз»ҷ JavaScript еј•ж“ҺпјҢзӯү JavaScript еј•ж“ҺиҝҗиЎҢе®ҢжҜ•пјҢжөҸи§ҲеҷЁеҶҚд»Һдёӯж–ӯзҡ„ең°ж–№жҒўеӨҚз»§з»ӯи§Јжһҗж–ҮжЎЈгҖӮ
